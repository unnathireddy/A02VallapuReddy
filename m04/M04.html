M04 Unnathi Sheep Ranch

This assignment introduces simplified scripting with jQuery.

The code is available at https://bitbucket.org/professorcase/M04.

Results of the unit tests are available at test/test.html.


Currently, the application gets the user's name and the size of their ranch. The app calculates square miles and estimates how many sheep they can hold on their land.


Your goal is to finish transitioning the app to jQuery and native JavaScript.


Modify at least one line of code in each of the following functions to use jQuery:

  1. getEstimate()
  2. showExample()
  3. addImage()
  4. Change the application name displayed in the menu to a suitable name for your application.

Click the button to start the app.
